1. Adjust settings in conf.js to match your desired settings.
2. Startup the vagrant box running the robo server.
  vagrant up                # start vagrant box
  vagrant ssh               # this logs into the vagrant box.
  cd /vagrant/robocoder/ActivityCoder     # to see the robocoder folder from git
  rails server              # starts server
3. Open the index.html file to run the test.
4. Check console.log for output.
